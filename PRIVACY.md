# App Code Privacy Policy

App Code is a local-first code editor. The app stores editor files, settings, Git remotes, terminal state, and runtime cache on the user device or in Apple-provided containers selected by the user.

## Data collection

App Code does not sell personal data and does not include third-party advertising tracking. This fork disables the upstream Judge0 remote execution panel by default.

## Sync

App Code stores remote host entries locally on the device. Secrets and Git credentials remain local/keychain-backed and are not intentionally mirrored through iCloud.

## User-controlled services

Users may connect to Git, SSH, SFTP, package registries, or other development services they configure inside the app. Those services are controlled by the user and governed by their own privacy policies.

## Contact

Open an issue at https://github.com/NightVibes33/app-code-ios/issues for privacy questions about this fork.
